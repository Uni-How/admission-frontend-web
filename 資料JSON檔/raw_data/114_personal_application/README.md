# 114 學年度個人申請原始資料

## 📁 資料來源
- **檔案**: `114_申請入學校系資料.xlsx`
- **來源目錄**: `crawler/可用資料/`
- **轉換日期**: 2025-12-05

## 📊 資料說明
此資料包含 114 學年度所有大學個人申請入學管道的校系資訊。

## 🔍 主要欄位（預期）
根據個人申請簡章，應包含以下欄位：
- 校名
- 校系代碼
- 系組名稱
- 招生名額
- 檢定標準（國文、英文、數學、社會、自然）
- 篩選倍率（第一階段）
- 採計科目與加權（第二階段）
- 同分參酌順序

## 📝 使用方式
